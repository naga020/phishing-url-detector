# Phishing URL Detector

A Python-based machine learning project to detect phishing websites using URL features.

## Features
- Extracts URL features: length, HTTPS, special symbols, suspicious words
- Random Forest classifier predicts Safe or Phishing
- Interactive testing using test_model.py

## Tech Stack
- Python, Pandas, Scikit-learn, Joblib

## How to Run
1. Train the model: `python train_model.py`
2. Test a URL: `python test_model.py`

## Dataset
- phishing_and_benign_websites.csv
